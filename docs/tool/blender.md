# blender3D编辑器

# 1. 快捷键

- `~`按键  切换视角

- `Shift` + `D`同时按住按键  复制当前选中的模型，随机移动位置

- `Shift` + `鼠标滚动键按下`移动当前视角平面的平面位置

-  `鼠标滚动键按下` 三维旋转当前的视图

-  `鼠标滚动` 放大缩小当前的视图

- `Tab`按键  当前模型进入、退出编辑模式

- `g`按键  移动当前模型的位置

- `x`按键  在`Shift + D`或者`g 按键`后，对当前模型进行x方向上移动位置

- `y`按键  在`Shift + D`或者`g 按键`后，对当前模型进行y方向上移动位置
